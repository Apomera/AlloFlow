# PasstheEPPP

Comprehensive study application for the Examination for Professional Practice in Psychology (EPPP).

## Features

- **📊 Dashboard** — Readiness score, study streaks, domain progress
- **🔬 Diagnostic Exam** — Full-length diagnostics with predicted score
- **📖 Study Mode** — Domain-by-domain content review with notes
- **🃏 Flashcards** — SM-2 spaced repetition with difficulty rating
- **❓ Quiz Mode** — Configurable quizzes with timer and flagging
- **📝 Practice Exam** — Timed 225-question exam simulation
- **📈 Analytics** — Performance trends and domain breakdowns
- **🎯 Study Goals** — Set and track weekly targets
- **💡 Memory Aids** — Mnemonics and study strategies
- **🔖 Bookmarks** — Save questions and topics for review
- **🔍 Search** — Full-text search across all content

## Content

- **2,100+ practice questions** across all 8 EPPP domains
- **120+ flashcards** with spaced repetition
- **Memory aids** and mnemonics for key concepts
- Enhanced rationales explaining why incorrect answers are wrong

## EPPP Domains

1. Biological Bases of Behavior (10%)
2. Cognitive-Affective Bases of Behavior (13%)
3. Social & Cultural Bases of Behavior (11%)
4. Growth & Lifespan Development (12%)
5. Assessment & Diagnosis (16%)
6. Treatment/Intervention (15%)
7. Research Methods & Statistics (7%)
8. Professional Ethics, Legal, & Professional Issues (16%)

## Tech Stack

- Vanilla HTML/CSS/JavaScript (no framework dependencies)
- Client-side only — all data stored in localStorage
- SM-2 spaced repetition algorithm
