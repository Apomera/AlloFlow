AAC & Visual Communication

Symbol Studio

AI-powered AAC board builder: generate unlimited custom PCS-style symbols with Imagen in 6 styles (flat vector, line art, cartoon, watercolor, bold comic, high-contrast B&W). Cloud sync across devices via Firebase.

Direct-Use AAC Mode

Full-screen communication board with tap-to-speak, a sentence strip for phrase building, and a session communication log exportable as CSV — ready for real AAC use in the classroom.

AI Word Prediction

After each symbol tap, Gemini predicts the 4 most likely next words and shows them as thumbnail chips above the grid — reducing the number of taps needed to build a message.

Type any sentence and AI maps each word to an AAC symbol category, checks the gallery for existing matches, and lets you review the mapping before adding symbols to a board in one click.

Export any board as a self-contained .html file with base64-embedded images, Web Speech API TTS, and a sentence strip. Opens on any device in any browser — no AlloFlow or internet required.

Single-switch scanning mode auto-highlights cells at 1–4 second intervals. Space bar or tap activates the highlighted symbol — designed for students who cannot independently point or tap.

Board Builder

AI-generates a complete word list from any topic, then generates images in batch. Customizable category color coding, 6 board themes (Pastel, Dark, High Contrast, Warm, Cool), and adjustable column count.

Print & Visual Schedules

Symbol Gallery

Architecture & Privacy

Google Education Infrastructure

Operates natively inside Gemini Canvas, leveraging your district's existing COPPA/FERPA-compliant Google Workspace security agreements.

Standard JSON "Sneakernet"

Because Gemini Canvas does not persist internal databases across sessions, AlloFlow relies on saving and loading curriculum directly as JSON files to your hard drive.

Frictionless Sharing

Distribute interactive lessons to students or collaborate with co-teachers simply by sending the JSON file and a canvas link.

Local-First Student Data

Student progress is tracked on-device, never uploaded. Teachers export data as JSON. The Student Portal uses codenames to keep identities private.

No Cloud Accounts Required

Zero third-party account creation for students. Bypasses the need to manage standalone application rosters.

Single-File Monolith

The entire application, including its assets, compiles into a single file payload, removing dependency vulnerabilities.

Offline HTML Bundles

Export units as standalone interactive packages that work without internet — perfect for homework equity.

Teacher Gate

Access code protection keeps student and teacher environments separate.

Privacy by Design

Student Entry Portal uses fun codenames (e.g. "Cosmic Badger") instead of PII.
