# Image shot-list — Fractions on the Number Line (Grade 3-4)

Generate in-app (Image tool) or any generator; attach to the noted resource. Alt text is
written to born-accessible standards — use it verbatim.

## 1. `fr-img-number-line-fourths` — attach to "The Space Between 0 and 1" (fr-reading)
**Prompt:** A large clean number line for 3rd graders running from 0 to 1, divided into four
equal segments with tick marks labeled 1/4, 2/4, 3/4 and endpoints labeled 0 and 1. Bright,
simple, generous spacing, no clutter.
**Alt text:** "A number line from 0 to 1 divided into four equal parts. The tick marks between
the ends are labeled one-fourth, two-fourths, and three-fourths."

## 2. `fr-img-fourths-vs-eighths` — optional, attach to the quiz
**Prompt:** Two stacked number lines of the same length from 0 to 1. The top is cut into 4
equal pieces with 1/4 marked; the bottom is cut into 8 equal pieces with 1/8 marked. A dotted
vertical line shows that 1/8 sits closer to 0 than 1/4. Clean and comparative.
**Alt text:** "Two number lines of equal length from 0 to 1. The top line is divided into four
parts with one-fourth marked. The bottom is divided into eight parts with one-eighth marked
closer to zero, showing that one-fourth is larger."

## 3. `fr-img-equivalent-half` — optional, attach to the concept sort
**Prompt:** One number line from 0 to 1 with two labels pointing at the SAME midpoint: 1/2
above the line and 2/4 below it, with a small caption "same place, two names." Friendly and
uncluttered.
**Alt text:** "A number line from 0 to 1 where a single midpoint is labeled one-half above the
line and two-fourths below it, showing they name the same location."
