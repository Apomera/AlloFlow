# 🧑‍🏫 Teacher Quick Reference Card

## Getting Started in 30 Seconds

1. **Open AlloFlow** → [Canvas link](https://gemini.google.com/share/c8baf80a46cc) or your self-hosted URL
2. **Paste your text** → Any article, chapter, or lesson content
3. **Click "Fullpack"** → Leveled text, glossary, quiz, and scaffolds generated at once
4. **Explore** → Use the toolbar to access specific tools

## Live Session Mode

**To start a live class:**
1. Click **"Go Live"** in the top bar
2. Share the **4-character code** with students
3. Students join at the session URL

**Live engagement modes:**
| Mode | Description |
|------|-------------|
| **Boss Battle** | Class works together — correct answers damage the boss |
| **Team Showdown** | Competitive team quiz with live leaderboard |
| **Democracy Mode** | Students vote; majority answer is submitted |
| **Escape Room** | Timed team puzzle challenges |
| **Diff Push** | Send different content to different student groups |

## Word Sounds Activity Types

| Activity | Students Practice |
|----------|-------------------|
| **Counting** | How many sounds in "cat"? → 3 |
| **Find Sounds** | First / middle / last phoneme isolation |
| **Blend Sounds** | /k/ /æ/ /t/ → "cat" |
| **Break Apart** | "dog" → /d/ /ɒ/ /g/ (Elkonin boxes) |
| **Rhyme Time** | What rhymes with "hat"? |
| **Sound Mapping** | Match phonemes to letters (grapheme bridge) |
| **Sight & Spell** | Orthography and spelling |
| **Word Families** | Group words by rime (-at, -ing, -ock) |

## Pro Tips

💡 **Check the Review Panel** before running Word Sounds — fix any AI phoneme errors first

💡 **Export HTML bundles** for students without reliable home internet

💡 **Toggle TeacherGate** to access BehaviorLens, Report Writer, and Symbol Studio

💡 **Ask AlloBot** (🤖) for contextual help — it knows which tool you're using

💡 **Use Fullpack** to generate everything at once rather than one tool at a time

💡 **Save as JSON** after a good session — reload it for any class section

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Generation not loading | Check internet; try Shift + Refresh (hard reload) |
| Audio not playing | Click anywhere on page first (browser requires a gesture) |
| Stuck inside a game | Press **Esc** to close and return to main view |
| AlloBot disappeared | Settings → Reset Onboarding to re-enable |
| Can't find a tool | Ask AlloBot or press **?** for Help Mode |
| Export failed | Try a smaller selection or a different format |

# 🎮 Student Quick Reference Card

## How to Join a Live Session

1. Go to the URL your teacher gives you
2. Enter the **4-letter code** your teacher puts on the board (like "ABCD")
3. Pick a **codename** (or get one assigned)
4. You're in! 🎮

---

## What the Icons Mean

| Icon | What It Is |
|:----:|------------|
| 📖 | Reading materials |
| ❓ | Quiz or questions |
| 🎮 | Games and activities |
| ⭐ | Your XP (experience points) |
| 🪙 | Gold you've earned |
| 🎒 | Your inventory |
| 🔊 | Click to hear it read aloud |
| 🤖 | AlloBot — your AI helper |

---

## Earning XP

| Action | XP Earned |
|--------|-----------|
| Correct quiz answer | +10–50 |
| Answer streak (3+ in a row) | Bonus XP! |
| Complete an activity | +25–100 |
| Help your team win | Varies |

Level up to unlock new items! 🛒

## Accessibility Options *(Settings ⚙️)*

If reading is hard, try these:

| Option | What It Does |
|--------|--------------|
| **Bigger text** | Makes all words larger |
| **Read aloud** 🔊 | Click any sentence to hear it |
| **Bionic mode** | Bolds part of each word to guide your eye |
| **Color overlay** | Changes the background color |
| **Different font** | Try OpenDyslexic for easier reading |
| **Dark mode** | Switches to a dark background |

## Adventure Mode Tips

🗡️ Your choices change the story — choose wisely!

💰 Save your gold for items that help you survive

🎲 Dice rolls add luck — higher numbers are better

🤝 In Democracy Mode, your vote counts for the whole class!

📖 At the end, your teacher can export your adventure as a storybook!

🏆 Get 3 in a row for a streak bonus!
